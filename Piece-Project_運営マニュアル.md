## 1. 管理モードの開け方

**方法A**: 画面左上のロゴ（🧩マーク）を**5回続けてタップ**
**方法B**: URLの末尾に `?admin=1` を付けて開く
　例: `https://michiyosi.github.io/peace-week/?admin=1`

→ パスワードを聞かれるので入力 → 画面下に**ADMINバー**が出ます。

**閉じ方**: ADMINバーの「× 管理モードを終了」

### ここだけ注意

**ページを再読み込みすると、パスワードだけ忘れられます。**
ADMINバーは出たままなのに、押すとパスワードを聞かれます。故障ではありません。
（パスワードをブラウザに保存しない作りにしているためです）
そのまま入力すれば、押した操作がそのまま続きます。

## 2. ボタンの説明

ADMINバーは4つのまとまりに分かれています。

### ■ 8月6日 本番（赤い枠）

| ボタン | すること |
|---|---|
| **📡 お披露目を全員に配信** | **当日の本番。押すと8秒後、会場の投影も、広場のスマホも、家で見ている人も、いっせいに演出が始まります。** 確認画面が出るので、落ち着いて「OK」を押してください。 |

### ■ ふだん見る

| ボタン | すること |
|---|---|
| 📊 統計 | 今何枚集まったか、日別・色別・テーマ別の内訳 |
| ⬇ CSV | 全投稿をExcelで開ける形で保存 |
| ⬇ JSON | 全投稿をデータ形式で保存（バックアップ用） |

### ■ 気になる投稿を消す

| ボタン | すること |
|---|---|
| 🗑 1枚ずつ | 押すと削除モードに入り、消したいピースをタップ→確認→非表示に |
| 🧹 まとめて | 「直近10分」などを選んで、その間の投稿をまとめて非表示に。**荒らされたときの復旧用** |

※どちらも「非表示」であって、データは消えません。間違えてもSupabaseから戻せます。

### ■ 準備・確認

| ボタン | すること |
|---|---|
| 🎨 絵のもとを抽出 | 集まったメッセージから、AIに渡す言葉を取り出す |
| ✨ 演出をひとりで確認 | **この画面の中だけ**で演出を再生。**配信されません。**確認用 |
| ⟲ 配信を待機に戻す | リハで何度も繰り返すときに使う。待機画面に戻ります |
| ↺ サーバと再同期 | 画面がおかしいときに、サーバの正しい状態を取り直す。**何も消えません** |

## 4. 困ったときは

### 押しても何も起きない

パスワードを聞かれていませんか。入力すれば続きが実行されます。

### 会場は光ったのに、スマホが光らない

**赤い「📡 お披露目を全員に配信」ではなく、金色の「✨ 演出をひとりで確認」を押した可能性があります。**
「⟲ 配信を待機に戻す」→ 会場PCを再読み込み → もう一度赤いボタンを押してください。

### ネットが落ちた／サーバが応答しない

会場の投影だけは、**単独で成立させられます**。
USBに入れた `index.html` をダブルクリックで開いてください。
ネットなしで動き、**Spaceキー**で演出が再生できます（パスワード不要）。
※この場合、オンラインで見ている人には届きません。

### 画面がおかしい

「↺ サーバと再同期」を押してください。何も消えません。

### 心ないメッセージが出た

「🧹 まとめて」で時間を指定して消すのが速いです。1枚だけなら「🗑 1枚ずつ」。

## 5. リハーサル用（USB）

`index.html` をUSBに入れて、会場PCで**ダブルクリック**するだけ。
ネット不要・ログイン不要・パスワード不要で、**810件のダミー**が入った状態で動きます。
赤い「リハーサル用」の札が出るので、本番と取り違えません。

| キー | すること |
|---|---|
| **Space** | 演出を再生（①から⑤まで通し） |
| **F** | 裏返して「みんなの想い」 |
| **R** | 最初からやり直す |
| **Enter** | 全画面 |
| **D** | **画面情報**（解像度・縦横比・向き・ブラウザ対応が一覧で出る） |

**Dキーは18時のリハで最初に押してください。** そのビジョンの素性が5分で分かります。
線の細さテスト（縞模様）も出るので、広場から見て何本目まで見分けられるかを控えてください。
