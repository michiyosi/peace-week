## コンセプト (v3.0〜 円形構造)

一人ひとりの「好きな色」と「2045年へのメッセージ」がピースになり、**中心の光（2045）から円になって外へ広がっていく**。810ピースがそろったら、みんなの回答をもとにした一枚の絵「2045年の広島」が **8/6のお披露目会で初公開**（このデモは集まっていく過程の可視化。絵の生成は回答が出揃ってから1回のみ、という運用方針）。キーワード:「広がる」「カラフル」「光」「パズルのピース」。

## つかいかた

色とメッセージをえらんで「ピースを置く」→ ピースが円の内側から順にはまる。ピースをタップするとその人のメッセージが読める。「8月6日、いっきにお披露目」で当日の演出（渦状に完成→光のリング）をプレビュー。

## 技術メモ

- 単一HTML。外部依存は Google Fonts のみ（Zen Maru Gothic / Yusei Magic）。Canvas2D 描画。
- 円形ジグソー: **15リング × (12,18,24,…,96)セクター = ちょうど810ピース**。リング内の隣接ピースはラジアル辺の「きのこ型」ノブ（対称プロファイル）で必ず噛み合う。リング境界は円弧。各リングはランダム位相で自然な見た目に。
- 埋まり順: 内側リングから外へ（リング内はランダム）= 「丸からどんどん広がる」を保証。
- 光の演出: 埋まった半径に追従する暖色グロー、中心コア（2045）、周囲を漂う装飾ピース、お披露目時の光リング＋紙吹雪。
- URLパラメータ: `?rm=1`（reduced-motion強制）/ `?seed=N`（初期ピース数）
- テストフック: `PW.place(色,msg,名)` / `PW.assemble()` / `PW.reset()` / `PW.fill(n)` / `PW.count()` / `PW.target` / `PW.version`
- 対応: モバイル / prefers-reduced-motion / IME変換確定 / Esc・focus-visible

## 更新手順

`index.html` を編集して main にコミットすると、GitHub Pages が自動再ビルド（数分）。反映確認は `?v=日付` を付けると確実。デプロイが失敗した場合は Actions から Re-run（または空コミット）。
